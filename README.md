# Project-1-College-Recruitment-Improvement-Study
Goal: Using Python, perform an exploratory data analysis of the CCM Computing Courses Entry Surveys. Provide insights that can help the Information Technologies department increase enrollment in computing classes. This repository contains the complete submission, including the VS_P1P3_Analysis.ipynb Jupyter Notebook, which holds all the code, plots, and analysis findings; the Major Survey Results - Fall 2024.csv file, which is the original dataset; the cleaned_majors_survey.csv file used for the study; and this README.md file, which describes the project's purpose and contents.
